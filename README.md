# 🔐 Password Strength Analyzer

A web application that analyzes password strength in real time.

## Features

-  Show/Hide Password
-  Live Strength Meter
-  Password Requirement Checklist
-  Estimated Crack Time
-  Responsive Cybersecurity-themed UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Future Improvements

- Password entropy calculation
- Secure password generator
- Copy password button
- Dark/Light mode
- Better crack time estimation
